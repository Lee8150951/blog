    CSS Modules

    Oreo! 2021/5/11 CSS Modules前端

    # 前言

    使用webpack.config.js开启CSS Modules功能：style-loader!css-loader?modules

    module.exports = {
  entry: __dirname + '/index.js',
  output: {
    publicPath: '/',
    filename: './bundle.js'
  },
  module: {
    loaders: [
      {
        test: /\.jsx?$/,
        exclude: /node_modules/,
        loader: 'babel',
        query: {
          presets: ['es2015', 'stage-0', 'react']
        }
      },
      {
        test: /\.css$/,
        loader: "style-loader!css-loader?modules"
      },
    ]
  }
};

    # 局部作用域

    产生局部作用的唯一方法就是使用一个独一无二的class名字，不会与其他选择器重名。

    在CSS Modules中，使用style.xxx可以将其编译为一个哈希字符串，如：

    CSS文件

    .title {
  color: red;
}

    App.js文件

    import React from 'react';
import style from './App.css';

export default () => {
  return (
    <h1 className={style.title}>
      Hello World
    </h1>
  );
};

    在以上中，将样式文件App.css输入到style对象中，然后引用{style.title}表示一个class名

    构建工具会将其编译为：

    <h1 class="_3zyde4l1yATCOkgn-DBWEL">
  Hello World
</h1>

    这就是将其编译成了哈希字符串，同样的App.css中的类名也会被同时编译为同名字符串

    # 全局作用域

    CSS Modules可以使用:global(.className)，在CSS文件中声明一个全局规则，凡是这样声明过的class，编译工具都不会将其编译为哈希字符串，如：

    .title {
  color: red;
}

:global(.title) {
  color: green;
}

    在需要引入该全局类时，直接用普通的class写法即可，就会自动引用全局class

    import React from 'react';
import styles from './App.css';

export default () => {
  return (
    <h1 className="title">
      Hello World
    </h1>
  );
};

    # Class组合

    CSS Modules中，一个选择器可以继承另一个类选择器的样式，称为composition

    关键字：composes

    在CSS文件中，通过在类中使用composes: 类名的方式，可以继承其他类中的属性

    .className {
  background-color: blue;
}

.title {
  composes: className;
  color: red;
}

    在App.js中，正常使用局部引入的方式使用即可

    import React from 'react';
import style from './App.css';

export default () => {
  return (
    <h1 className={style.title}>
      Hello World
    </h1>
  );
};

    于是，.title类中会自动引入.className这个类中的所有属性，而h1会被编译为：

    <h1 class="_2DHwuiHWMnKTOYG45T0x34 _10B-buq6_BEOTOl9urIjf8">

    # 其他模块的使用

    在CSS Modules规则中CSS可以使用composes关键字引入其他CSS文件的类，如：

    another.css

    .className {
  background-color: blue;
}

    App.css

    .title {
  composes: className from './another.css';
  color: red;
}

    # 补充

    CSS Modules 支持使用变量，不过需要安装 PostCSS 和 postcss-modules-values
