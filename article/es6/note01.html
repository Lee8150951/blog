    解构赋值

    Oreo! 2021/4/13 ES6前端

    # 概述

    ES6新增语法，对原本赋值语法的一种拓展。通过解构赋值将属性、值从对象、数组中取出，赋值给其他变量。

    解构赋值由两部分组成：

    • 解构的源，解构赋值表达式的右边部分
    • 解构的目标，解构赋值表达式的左边部分

    # 数组解构

    # 1、标准

    let [a, b, c] = [1, 2, 3]

    以上代码可以理解成：JavaScript将结构目标拆分成a、b、c三个，并对应解构源进行相应赋值

    这是最为基本的数值解构赋值

    # 2、嵌套

    数组的解构赋值内还可以出现嵌套，如：

    let [a, [[b], c]] = [1, [[2], 3]]

    也就是说：只要解构的目标和源格式一一对应，解释器会自动对应并相应赋值

    # 3、省略

    对于不明了的解构目标，可以通过省略的方式对其进行解构赋值

    let [a, , b] = [1, 2, 3]

    如上，a和b变量将被分别赋予1和3，但是由于中间值的缺失，解构源的2将不被赋予任何目标

    # 4、字符串

    通常来说在JavaScript中，可以将字符串当作一种特殊的数组来处理，甚至其可以使用数组的方法

    字符串作为一种可遍历对象，也可以参与解构赋值运算

    let [a, b, c, d, e] = 'hello'
// a = 'h'
// b = 'e'
// c = 'l'
// d = 'l'
// e = 'o'

    # 5、解构默认值

    在解构赋值中可以在结构目标中对目标进行赋值，这个地方的赋值被称作默认值

    注意：当解构模式有匹配结果，且匹配结果是 undefined 时，会触发默认值作为返回结果

    let [a = 3, b = a] = []
// a = 3, b = 3
let [a = 3, b = a] = [1]
// a = 1, b = 1
let [a = 3, b = a] = [1, 2]
// a = 1, b = 2

    # 6、剩余运算符

    let [a, ...b] = [1, 2, 3];
// a = 1
// b = [2, 3]

    关于三点运算符的方法会另外学习

    # 7、特别用法：交换变量

    在一个解构表达式中可以交换两个变量的值

    没有解构赋值的情况下，交换两个变量需要一个临时变量

    var a = 1
var b = 3

[a, b] = [b, a]
console.log(a)
// 3
console.log(b)
// 1

    # 对象解构

    # 1、标准

    和数组的标准方式一样，对象的结构也是一一对应的方式，但是值得注意的是：对象的解构赋值通过的是键值对的“键”匹配进行的

    var o = {p: 42, q: true}
var {p, q} = o
// p = 42
// q = true

    # 2、无声明赋值

    var a, b
;({a, b} = {a: 1, b: 2})

    注意：赋值语句周围的圆括号 ( ... ) 在使用对象字面量无声明解构赋值时是必须的

    {a, b} = {a: 1, b: 2} 不是有效的独立语法，因为左边的 {a, b} 被认为是一个块而不是对象字面量

    然而，({a, b} = {a: 1, b: 2}) 是有效的，正如 var {a, b} = {a: 1, b: 2}

    ( ... ) 表达式之前需要有一个分号，否则它可能会被当成上一行中的函数执行

    # 3、新变量赋值

    可以从一个对象中提取变量并赋值给和对象属性名不同的新的变量名

    var o = {p: 42, q: true}
var {p: foo, q: bar} = o

console.log(foo)
// 42
console.log(bar)
// true

    # 4、解构默认值

    和数组的解构类似，对象解构时变量可以先赋予默认值

    当要提取的对象对应属性解析为 undefined，变量就被赋予默认值

    var {a = 10, b = 5} = {a: 3}

console.log(a)
// 3
console.log(b)
// 5
