    # 一、语言基础（下）

    # 1.3数据类型

    # 1.3.1String类型

    字符串可以使用双引号（""），单引号（''）以及反引号（``）

    ES语法中字符串是不可变的，其含义是一旦创建它的值就不能再进行改变，如果要修改某个变量中的字符串值，必须先销毁原始的字符串，然后将包含新值的另一个字符串保存到该变量中

    将一个值转换为字符串有两种方法：toString()以及String()方法、

    toString()方法可以用于数值、布尔值、对象和字符串变量，但是null和undefined值没有该方法，其唯一的作用就是返回当前值的字符串等价物，如：

    let age = 10;
let ageStr = age.toString(); // "10"
let found = true;
let foundStr = found.toString(); // "true"

    toString()方法是可以接受参数的，它接受一个底数参数，即用底数来输出数值的字符串表示，如二进制、八进制、十六进制等等，如：

    let num = 10;
console.log(num.toString()); // "10"
console.log(num.toString(2)); // "1010"
console.log(num.toString(8)); // "12"
console.log(num.toString(10)); // "10"
console.log(num.toString(16)); // "a"

    另外，如果不确定一个值是否是null或者undefined时，可以使用String()转型函数，它使用会返回表示相应类型值的字符串，且遵循如下规则：

    • 如果该值有toString()方法，则调用该方法并返回结果
    • 如果值是null，则返回"null"字符串
    • 如果只是undefined，则返回"undefined"字符串

    ES语法中通过模板字面量可以实现字符串插值，也就是可以在一个连续定义的中插入一个或多个值，通过${}实现

    let value = 5;
let exponent = 'second';
// 模板字面量的字符串插值
let interpolatedTemplateLiteral = `${value} to the ${exponent} power is ${value * value}`

    注意：插值表达式中可以调用函数和方法

    # 1.3.2Symbol类型

    Symbol（符号）类型是ES6新增的数据类型，符号是原始值，且符号实例是唯一、不可变的。

    其用途就是确保对象的属性使用唯一标识符，不会发生属性冲突的危险。

    Symbol类型就是用来创建唯一记号的

    原始数据类型 Symbol ，表示独一无二的值，最大的用法是用来定义对象的唯一属性

    # 1）基本使用

    符号需要使用Symbol()函数进行初始化操作，因为符号本身是原始类型，所以typeof操作符对符号返回symbol

    let sym = Symbol();
console.log(typeof sym); // symbol

    在调用Symbol()函数时，也可以传入一个字符串参数作为对符号的描述（description）

    描述的作用在于调试代码，但是这个字符串参数与符号的定义和标识完全无关

    let sym1 = Symbol('foo');
let sym2 = Symbol('foo');
console.log(sym1 == sym2); // false

    只要创建Symbol()实例，并将其用作对象的新属性，就可以保证它不会覆盖已有的对象属性，无论是符号属性还是字符串属性

    最重要的是，Symbol()函数不能够和new关键字一起作为构造函数的使用，为了避免创建符号包装对象

    # 2）使用全局符号注册表

    如果运行时的不同部分需要共享和重用符号实例，那么可以用一个字符串作为键，以在全局符号注册表中创建并重用该符号

    需要使用到的方法是：Symbol.for()

    let fooGlobalSymbol = Symbol.for('foo');
let otherFooGlobalSymbol = Symbol.for('foo');

console.log(fooGlobalSymbol == otherFooGlobalSymbol); // true

    另外，还可以使用Symbol.keyFor()方法来查询全局注册表

    // 创建全局符号
let s = Symbol.for('foo');
console.log(Symbol.keyFor(s)); // foo

// 创建普通符号
let s2 = Symbol('bar');
console.log(Symbol.keyFor(s2); // undefined
    # 3）使用符号作为属性

    凡是可以使用字符串或数值作为属性的地方，都可以使用符号

    对象字面量只能在计算属性语法中使用符号作为属性

    因为符号属性是对内存中符号的一个引用，所以直接创建并作属性的符号不会丢失

    # 4）使用场景

    由于每一个 Symbol 的值都是不相等的，所以 Symbol 作为对象的属性名，可以保证属性不重名

    // 定义符号
let sy = Symbol("key1");
 
// 写法1
let syObject = {};
syObject[sy] = "kk";
console.log(syObject);    // {Symbol(key1): "kk"}
 
// 写法2
let syObject = {
  [sy]: "kk"
};
console.log(syObject);    // {Symbol(key1): "kk"}
 
// 写法3
let syObject = {};
Object.defineProperty(syObject, sy, {value: "kk"});
console.log(syObject);   // {Symbol(key1): "kk"}

    注：Object.defineProperty() 方法会直接在一个对象上定义一个新属性，或者修改一个对象的现有属性，并返回此对象

    # 1.4操作符

    # 1.4.1一元操作符

    只操作一个值的操作符称为一元操作符

    # 1）递增、递减

    递增：++

    递减：--

    递增递减的一元操作符有两种版本，分别是：前缀和后缀

    前缀写法如下所示：

    let age = 20;
++age;
// 等价于
age = age + 1;

let age = 20;
--age;
// 等价于
age = age - 1;

    注意：无论使用前缀递增还是前缀递减操作符，变量的值都会在语句被求值之前改变（被成为副作用）

    前后缀写法是非常类似的，只不过放在操作对象的后面

    前缀和后缀最重要的区别在于：后缀递增和递减在语句求值之后才发生

    # 2）一元加减
    let num = 25;
num = +num; // 25
num = -num; // -25

    一元加减主要用于基本的算术，但同时也可以用于数据类型的转换
